# CKB08 Website Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build a Next.js website for Cannstatter Kurve Berlin e.V. with public pages, blog, termine, contact form, and an admin panel for content management.

**Architecture:** Next.js 15 App Router on Vercel Free. Turso SQLite via Drizzle ORM for data. NextAuth.js v5 Credentials for admin auth. Tiptap WYSIWYG for blog editing. Vercel Blob for image uploads. All German, Swabian dialect where fitting.

**Tech Stack:** Next.js 15, Turso/libsql, Drizzle ORM, NextAuth.js v5, Tailwind CSS, Tiptap, Vercel Blob, bcryptjs, isomorphic-dompurify

**Spec:** `docs/superpowers/specs/2026-04-02-ckb08-website-design.md`

**Security note:** Blog content is rendered as HTML from Tiptap. Although only authenticated admins can create content, all HTML output is sanitized with DOMPurify before rendering to prevent XSS.

- [ ] **Step 1: Create NextAuth configuration**

Create `src/lib/auth.ts`:

```typescript
import NextAuth from "next-auth";
import Credentials from "next-auth/providers/credentials";
import { compare } from "bcryptjs";
import { db } from "./db";
import { users } from "./schema";
import { eq } from "drizzle-orm";

export const { handlers, signIn, signOut, auth } = NextAuth({
  providers: [
    Credentials({
      credentials: {
        email: { label: "Email", type: "email" },
        password: { label: "Passwort", type: "password" },
      },
      async authorize(credentials) {
        if (!credentials?.email || !credentials?.password) return null;

        const email = credentials.email as string;
        const password = credentials.password as string;

        const [user] = await db
          .select()
          .from(users)
          .where(eq(users.email, email))
          .limit(1);

        if (!user) return null;

        const isValid = await compare(password, user.passwordHash);
        if (!isValid) return null;

        return { id: user.id, email: user.email, name: user.name };
      },
    }),
  ],
  pages: {
    signIn: "/admin/login",
  },
  session: {
    strategy: "jwt",
  },
  callbacks: {
    async jwt({ token, user }) {
      if (user) {
        token.id = user.id;
      }
      return token;
    },
    async session({ session, token }) {
      if (session.user) {
        session.user.id = token.id as string;
      }
      return session;
    },
  },
});
```

"use server";

import { db } from "@/lib/db";
import { contactMessages } from "@/lib/schema";
import { ulid } from "ulid";

export async function submitContactForm(formData: FormData) {
  const name = formData.get("name") as string;
  const email = formData.get("email") as string;
  const nachricht = formData.get("nachricht") as string;

  if (!name || !email || !nachricht) {
    return { error: "Bitte alle Felder ausfüllen." };
  }

  if (!email.includes("@")) {
    return { error: "Bitte eine gültige E-Mail-Adresse eingeben." };
  }

  await db.insert(contactMessages).values({
    id: ulid(),
    name,
    email,
    nachricht,
  });

  return { success: true };
}

export function sanitizeHtml(dirty: string): string {
